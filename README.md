# Video Belajar

A platform for interactive video learning, built with **Vanilla HTML** and **Tailwind CSS**.

## Project Structure

- `index.html`: The main landing page.
- `login.html`: The login page.
- `register.html`: The registration page.
- `assets/`: Directory for storing images and logos.

## How to Run

Simply open any of the `.html` files in your web browser. No installation or build steps are required.

## Customization

- **Styling**: Tailwind CSS is loaded via CDN in the `<head>` of each HTML file.
- **Colors**: Custom colors (`primary`, `secondary`, etc.) are defined in the `tailwind.config` script within each HTML file.
- **Images**: Place your images in the `assets/` folder and reference them in the HTML (e.g., `<img src="assets/my-logo.png">`).
